    十年之约，长白山

    冰洋 10/7/2018 旅行TODO

    国庆长假，说走就走的旅行！超多图预警。

    # 决定

    之前考虑过去长白山，但是没有机会，工作原因也没有太多到时间去考虑这趟长途旅行。 很突然的决定，8月末，朋友老杨私聊我，要不要去长白山，马上可以抢十月一的票了！老杨老家就在长白山脚下，和我说过他们美丽到小镇，还说过2015年很多游客共赴十年之约到盛况，描述过冬天漫天飞雪到风光...很是羡慕。
    工作之余，果断，开始了抢票准备。不经意抢到了通往长春到卧铺票，开始了出发前的计划。同时拿出刚买到佳能，去接小哥
    期间老杨说长白山玩一天，小镇玩一天，其他时间会很长，于是乎，我各种马蜂窝，论坛到差找旅游攻略。想去别处逛逛，买了去安图到车票，还打算去漠河看看，各种计划，具体行程。但是忽略了一个问题，天气还有长时间到换成交通工具带来到不方便。
    中秋节回家一趟，吃个团圆饭，国庆就可以没心没肺到出去疯了。有个不好到消息就是9月底长白山的确连续下了两天雨，长白山顶下起了暴雪。会封山。

    # 出发

    9月29日晚下班，匆匆见了一个有趣到人， 然后辗转北京站。和老杨见了面，老杨比我车早，老杨先出发，晚上十点12坐上了通往长春到车。上铺有些不舒服，头顶也感觉发凉，夜里醒了几次。一夜无趣，30号早上6点半左右，列车到了长春。和朋友约定好站前集合，正式开始了长白山之旅。 在长春转了转，感受了下长春点地铁。站台很漂亮，节奏很慢 去了躺人民广场，长春天气不是很好，有点凉，里面穿短袖，外面套了个外套 收起相机，等到了老杨的同事大斌，此行的另一人，他是辗转两辆列车到的长春。因为老杨要回家，长白山天池可以陪我，此行大多时间要和大斌度过了。
    下午一点左右，出发去敦化。动车，窗户旁边，看到了初秋的森林，河流，一路经过很多穿山隧道，视角不好也没留下照片。 1个半小时形成，来到了敦化站。（当时忘拍照了，此图来源网络，现在敦化站很漂亮，有个广场，很多花草） 所有标示都是朝汉双语了 天气一直阴着，预示着行程不会很顺利。下车没赶上直达二道白河镇的车，辗转到永庆，再坐车到二道白河。上车开往二道百合，一路阴雨，很快就天黑了，路上聊着长白山已经封山，然后累的睡着了。 晚上7点左右到了二道白河，下车雨下的很大了，但是看到美丽的雨中小镇，还是很兴奋。 雨中漫步，去了一家牛肉汤饭馆。天下着雨，体感很凉了，喝完牛肉汤暖暖身子，甚是舒服。 之后，雨中逛了下超市。雨天去哪玩，包括长白山网吧一日游都考虑过？？最后初步计划第二天就在小镇雨中逛逛吧。 来到宾馆，看着窗外的雨。和有趣的大斌讨论了下，雨天到底去哪玩好。的确是累了，也没个明白计划，睡找了。

    # 延吉之行

    早上我六点就醒了，刷着手机，想着去哪玩。然后和没睡醒的大斌说这两天不能就这么呆着，要不要去延吉逛逛。一拍即合，和老杨说了声，立刻赶往汽车站，坐上了通往延吉点汽车。
    一路颠簸，不过沿途美不胜收。中途特别想下车，感受下森林，田野，河流。 颠簸四个小时，终于可以下车了 延吉市区 来到一个火车站附近点一个小宾馆，五洲宾馆，老板娘是朝鲜族的，很热情。
    然后稍微计划了下这两天行程，帽儿山，朝鲜民俗风情院，等等。。。

    # 帽儿山

    吃了个回锅肉打车来到帽儿山 免费景区，人挺多的，一路原始林木，雨后空气特别新鲜，感受大自然。 鹦鹉算命 秋天的森林 山不是很高，很快来的了山顶，观光台 山顶的许愿树 山不是很高，山顶能够俯视延吉整个城区，很美。

    # 朝鲜民俗风情园

    匆匆下山，没有土著带领，没有准备充分。来到了一个未完工的民俗园。传统的朝鲜民族聚居地应该在临近朝鲜的地方，时间有限，没有去。 落日

    # 延吉大学城

    延吉号称东北的成都，就是吃的多，延吉市区这块玩能玩一天，但是吃至少三天。但是作为两个对吃毫无研究的单身男，对此毫无计划。于是来到了延吉大学，体验延吉的夜生活。 中途公交路过布尔哈通河上的桥，看到夕阳下的布尔哈通河，没到站，果断下车了。 徒步走了20分钟，开到延吉大学城 晚上的延吉大学很安静，几乎没人，溜达了一圈。去了一家韩式烤肉店。特别火爆，老板很热情，虽然等了一个多小时才吃上饭。。 点的套餐，韩式烤肉，味道不错，可涮可烤，还有个辣炒年糕条还是啥来着，超级辣。米酒没了，可惜了。 吃完饭，沿着布尔哈通河溜达了一圈

    # 延吉漫展

    第二天上午，准备去眼镜博物馆逛逛。公交车上。看到延吉会展中心很热闹，于是就突发奇想下车。误打误撞开始漫展之旅。。 帅气的宝儿姐，忽略旁边一脸尴尬的大斌 人太多，再就是不太会拍人像，没有拍太多。

    # 民松

    看完漫展，找了家民松朝鲜餐厅，坐着吃，餐厅民族风，挺不错点。点了几个朝鲜特殊美食。朝鲜冷面，干锅拌饭，蔬菜沙拉，味道不错米酒特别好喝，酸甜，冰镇，很爽口

    # 延吉博物馆

    延吉博物馆讲述了延吉地区民俗，历史，文化等 朝鲜服装秀 时间比较赶，逛完博物馆。就赶忙赶往汽车站，前往二道白河，准备天池之旅

    # 长白山

    # 二道白河

    只能遇到晚上的二道白河 晚上在白河小镇骑行，感受大自然，不用担心旁边森林熊出没😄

    # 长白山

    提前买好票，早上5点起床洗漱，吃完饭，出发！
    吃完饭，先到长白山景区旅游车站 天气不错，祈祷可以上天池。听老杨说，上午应该景区能开 做大巴车到景区脚下集散中心。大概40分钟车程，沿途原始森林。 到达，长白山脚下。 然后换乘景区巴士，正式开始长白山之旅。沿途雪慢慢变多，前几天山顶大雪，需要清理道路所以才会封山。 因为天池景区暂时关闭，先去点长白瀑布景区 雪山温泉瀑布，美不胜收
    一路攀爬，终于到了长白瀑布。因为安全，只能里离很远点地方看，瀑布不是很大，但是水流清澈湍急。 换乘 来到换乘点，包车上山，因为长白山长年积雪，山高，没有人行上山道，只有一条曲折点环山路至上山顶观景台，所以这段路要包车上山。 一路颠簸格外刺激。安全到达山顶。下车后，瞬间被广雪山和远处广袤的森林震撼到了 开始前往天池观景台 神圣的天池（相机感光片有个灰尘，没带清理工具，一直有个黑点▪️，非常难受） 山顶的风景 山顶的游客们 山下还有几个景点，山上人也越来也多。下山，非常依依不舍。⬅坐车下山，刺激。

    # 小天池和绿渊潭

    地下森林关闭，最后去的小天池和绿渊潭。待续。。。
